# 工资管理系统功能与数据库说明

> 本文档合并说明：
> 1. 与原始数据库脚本 `salary_management(1)(1).sql` 相比，当前系统数据库结构的全部变化；
> 2. 系统的详细功能介绍与技术实现。

---

## 一、项目概述

### 1.1 系统定位
本系统是一个面向中小型企业的**工资管理信息系统**，覆盖部门、职位、员工、考勤、请假、加班、绩效、工资、奖金、扣款、职位变动、项目参与、支付方式及系统公告等核心业务，并提供基于角色的权限控制。

### 1.2 技术栈

| 层级 | 技术 |
|---|---|
| 后端 | Spring Boot 3.2 + Spring Security + Spring Data JPA + JWT |
| 数据库 | MySQL 8（默认库名 `salary_management`） |
| 缓存/限流 | Redis + Bucket4j |
| 前端 | Vue 3 + TypeScript + Vite + Tailwind CSS + Lucide 图标 |
| 部署 | Spring Boot 单端口托管前端 + Cloudflare Tunnel（域名 `https://salary.feiyu.rest`） |

### 1.3 运行方式

```bash
# 后端
mvn clean package -DskipTests
java -jar target/salary-management-1.0.0.jar --spring.profiles.active=dev

# 前端（开发）
npm install
npm run dev

# 前端（生产构建并集成到后端）
npm run build
cp -r dist/* ../backend/src/main/resources/static/
```

## 二、数据库结构变化说明

| nickname | VARCHAR(50) | 昵称 |

| avatar | VARCHAR(255) | 头像 URL |

完全新增，用于顶部铃铛公告：

| 字段 | 类型 | 说明 |

## 三、系统功能详细介绍

### 3.1 认证与权限

#### 3.1.1 登录与注册
- 用户通过 `username` + `password` 登录，后端返回 JWT Token。
- 支持自助注册（开发环境默认开启，可通过 `application-prod.yml` 关闭）。
- 密码使用 `BCryptPasswordEncoder` 加密存储。

#### 3.1.2 角色与权限
- `ADMIN`：可访问所有菜单、管理用户、发布系统公告、审批工资等。
- `USER`：普通员工，只能查看与自己相关的数据（考勤、请假、工资、绩效等）。

#### 3.1.3 账号状态
- 管理员可在“用户权限”页面启用/禁用账号。
- 管理员账号不可被禁用，避免系统锁死。

### 3.2 个人资料与头像

- 用户在“个人资料”页可修改昵称、用户名、密码。
- 支持上传头像（JPG/PNG/GIF/WebP）。
- 上传成功后，资料页与顶部导航栏的头像会同步更新，并持久化到 `localStorage`。
- 头像 URL 示例：`/api/user/avatar/{uuid}.jpg`，通过域名访问正常。

### 3.3 部门管理

- 维护企业部门树：部门代码、部门名称、上级部门、部门负责人。
- 支持新增、编辑、删除部门。
- 删除前校验是否被职位引用，防止误删。

### 3.4 职位管理

- 职位与部门绑定，包含基本工资。
- 支持职位的新增、编辑、删除。
- 员工变动职位时，会触发职位变动记录。

### 3.5 员工管理

- 维护员工基本信息：姓名、性别、出生日期、入职日期、在职状态、职位、部门。
- 支持新增、编辑、删除员工。
- 删除员工时级联删除关联的考勤、请假、加班、工资、奖金、扣款、支付方式等记录。

### 3.6 员工综合管理

- 以员工维度集中管理其奖金与罚款。
- 奖金/罚款按计薪周期记录，支持备注。
- 界面展示奖金合计、扣款合计，布局清晰。

### 3.7 考勤管理

- 记录员工每日考勤状态：`正常`、`迟到`、`早退`、`缺勤`。
- 记录迟到分钟数、早退分钟数、缺勤标记。
- 按日期查询，支持统计。

### 3.8 请假管理

- 记录请假类型、起止日期、请假天数、审批状态。
- 审批状态：`PENDING`、`APPROVED`、`REJECTED`。
- 工资计算时会根据请假天数扣减工资。

### 3.9 加班管理

- 记录加班日期、加班时长、审批状态。
- 审批通过后计入工资加班费。

### 3.10 绩效考核

- 记录员工绩效等级（A/B/C/D 等）与考核周期。
- 考核周期使用文本输入，如 `2026-06`，提升使用体验。
- 绩效结果影响工资中的绩效奖金。

### 3.11 职位变动

- 记录员工从原职位调动到新职位的过程。
- 包含：员工、原职位 ID、新职位 ID、变动日期、变动原因。
- 系统初始化时会自动生成 30+ 条职位变动演示数据，确保原职位/新职位不为空。

### 3.12 工资管理

- 工资记录包含基本工资快照、各类奖金与扣款明细、应发合计、扣款合计、实发工资。
- 工资状态流转：`GENERATED`（已生成）→ `APPROVED`（已审批）→ `PAID`（已发放）。
- 支持按员工、月份查询工资记录。
- 提供工资明细弹窗，展示构成详情。

### 3.13 奖金与扣款

- 奖金和扣款均与员工、计薪周期绑定。
- 支持在工资生成前录入，或事后调整。
- 工资计算时自动汇总到工资记录中。

### 3.14 项目管理

- 维护项目基本信息：项目名称、项目状态。
- 维护项目成员及角色、贡献系数。
- 员工可参与多个项目，项目可包含多个员工（M:N）。

### 3.15 支付方式

- 每个员工绑定一种支付方式：银行卡、支付宝、微信。
- 记录账号/卡类型，用于工资发放。

### 3.16 系统公告

- 顶部导航铃铛图标进入公告面板。
- 管理员可发布公告、置顶公告、删除公告。
- 普通员工可查看公告列表。
- 公告包含标题、内容、发布人、发布时间、置顶状态。
- 系统初始化时自动生成节日/祝福类公告（如端午节公告）。

### 3.17 交互体验优化

- **提示消息**：使用 Vue Teleport 挂载到 body，避免被顶部导航遮挡。
- **表格网格线**：加深表头与单元格边框，提升可读性。
- **日期选择器**：绩效考核、员工综合管理等页面的月份输入改为文本框，支持 `YYYY-MM` 直接输入。
- **系统设置**：删除冗余说明文字，界面更简洁。
- **操作栏**：修复按钮缺失/显示异常问题。

### 3.18 域名对外访问

- 通过 Cloudflare Tunnel 将域名 `https://salary.feiyu.rest` 映射到本地 Spring Boot 服务（端口 8088）。
- Spring Boot 同时托管后端 API 和前端静态资源（`backend/src/main/resources/static`）。
- 前端构建后复制到后端 static 目录，实现单端口部署。
- 跨域配置允许指定域名访问，开发环境默认允许所有来源。
- `SecurityConfig` 已放行前端静态资源，避免域名访问出现 403。

## 五、部署建议

1. **数据库**：执行 `backend/sql/init.sql` 初始化，或让 JPA `ddl-auto=update` 自动建表（已有数据时建议先备份）。
2. **环境变量**：生产环境务必通过环境变量覆盖 `JWT_SECRET`、`DB_PASSWORD`、`CORS_ALLOWED_ORIGINS`。
3. **静态资源**：每次前端代码更新后，执行 `npm run build` 并将 `dist/` 内容复制到 `backend/src/main/resources/static/`。
4. **隧道**：启动 `cloudflared tunnel --config backend/cloudflare/config.yml run`，确保 service 指向 `http://127.0.0.1:8088`。
5. **管理员账号**：首次启动后请尽快修改默认管理员密码。
